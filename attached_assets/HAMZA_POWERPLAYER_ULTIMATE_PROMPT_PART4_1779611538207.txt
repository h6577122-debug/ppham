  • Create a SINGLE canvas manager object in JS:
      const HP = {
        canvases: {},
        rafs: {},
        pause: (id) => cancelAnimationFrame(HP.rafs[id]),
        resume: (id) => { /* restart RAF loop */ }
      }

  • Each canvas has its own RAF loop stored in HP.rafs
  • IntersectionObserver pauses each canvas when its section leaves viewport
  • This prevents all 6 canvases running at full speed simultaneously

  • Everything wrapped in DOMContentLoaded
