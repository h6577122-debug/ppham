  • Airplane canvas: separate layer (own RAF or shared with particles)
  • Physics canvas: own RAF (separate from background)

  • DNA helix: simplified (single strand)

  • Physics simulation PAUSES when section is NOT visible
  • Ticker pauses when not in viewport (CSS animation-play-state)
  • Code rain pauses when terminal section is not visible
