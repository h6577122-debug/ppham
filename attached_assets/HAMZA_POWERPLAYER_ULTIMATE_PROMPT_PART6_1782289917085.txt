    Total transition: 1.1 seconds, absolutely smooth

    like flipping a physical card, revealing the Urdu content on the "back"
  • Language toggle button: small pill in nav — [EN | اردو]

EFFECT 44: MAGNETIC SECTION SNAPPING
  • When user scroll velocity drops below a threshold near a section boundary:
    The page gently "snaps" to the nearest section with a smooth deceleration
  • Implemented with custom scroll physics:
    Track scroll velocity via RAF + requestAnimationFrame
    When velocity < 15px/frame AND within 120px of section boundary → snap
    The snap uses a custom spring easing (not CSS scroll-snap which is too rigid)
  • The snap is SUBTLE — feels like the page is alive and breathing
    It should not feel like forced scroll snapping, just like gentle assistance
  • Does NOT apply when user is actively scrolling fast (velocity > 15px/frame)
  • Can be disabled via a small toggle in the corner: [FREE SCROLL | SNAP MODE]

EFFECT 45: APP ICON 3D STACK
  • In the Trophy Room section (Section N):
    Each app's icon is displayed as a 3D icon stack (like iOS stacked cards)
    3 identical icons stacked with 6px offset + subtle shadow between layers
    Stack depth gives a physical, tactile feeling
  • Hover any stack: the top card slides UP (translateY -40px) and rotates 15°
    Revealing a secondary card with the app's download count and rating
    The secondary card also slides: offsetting at 20° showing a third card
    with the app's Play Store link button
  • Each icon is rendered as a rounded-square CSS shape with gradient background
    matching the app's actual color scheme (defined per-app)
    No image files — all CSS gradient art + app initial letter in Orbitron Bold

EFFECT 47: TESTIMONIAL CARD FLIP REVEAL
  • The testimonials in Section R now use a 3D card flip system
  • Each testimonial card: identical front and back
    Front: Blurred/obscured — only shows the reviewer's country flag + star rating
    Back (after flip): Full review, reviewer name, app they reviewed
  • Cards are FACE DOWN on load
  • On scroll into viewport: cards flip face-up in a cascading sequence
    Card 1 flips at 0ms, Card 2 at 150ms, Card 3 at 300ms, etc.
  • The flip uses CSS rotateY with perspective(600px) — looks physically real
  • Manual interaction: clicking any card flips it back face-down, then face-up
  • The "suspense" of the face-down reveal makes each testimonial feel like a prize

EFFECT 48: AMBIENT BREATHING BORDER
  • All major glass cards now have a new ambient glow border that BREATHES
  • The border glow oscillates between:
    Dim: rgba(0,240,255,0.08) border + 0px spread
    Bright: rgba(0,240,255,0.25) border + 12px spread
  • Each card has a DIFFERENT breathing speed and phase (asynchronous):
    Card 1: 3.2s cycle  |  Card 2: 4.1s cycle  |  Card 3: 2.8s cycle
    This prevents synchronized "pulsing" (which looks fake/cheap)
    The slight phase difference makes the page feel genuinely alive
  • On hover: breathing pauses and border goes to maximum brightness instantly
